# Merath Calculator - Design System

## Design System Overview
Professional, culturally appropriate design system for Islamic inheritance calculator.

## Color Palette
- **Primary**: Islamic Green (#2e7d32)
- **Secondary**: Professional Blue (#4f9eff)
- **Tertiary**: Warm Gold (#ffa500)
- **Neutrals**: 8-step gray scale
- **Semantic**: Success, Warning, Error, Info

## Typography
- **Arabic**: Cairo (Google Fonts)
- **English**: Plus Jakarta Sans (Google Fonts)
- **Scale**: Display (32-24px), Headline (20-16px), Body (16-12px)

## Components
- Buttons (Primary, Secondary, Tertiary, Outline, Ghost, Danger)
- Cards (4 elevation levels)
- Inputs, Modals, Navigation

## Spacing System
- 8-point grid: 4px to 48px
- Consistent padding and margins

## Borders
- Hairline (1px), Thin (2px), Medium (3px), Thick (4px)

## Accessibility
- WCAG AA compliant color contrast
- Keyboard navigation support
- ARIA labels for screen readers
- Touch targets ≥44px
