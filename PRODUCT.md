# Merath Calculator - Product Overview

## Product Name
Merath Calculator

## Description
A comprehensive Islamic inheritance calculator app that helps users calculate inheritance shares according to Islamic law (Shariah) across different madhabs (Hanafi, Maliki, Shafi'i, Hanbali, and Jafari). The app provides a user-friendly interface for entering estate details, calculating shares, and generating professional PDF reports.

## Target Audience
- Muslims seeking to understand inheritance distribution
- Islamic scholars and students
- Legal professionals dealing with Islamic inheritance matters
- Families planning estate distribution

## Key Features
- Multi-madhhab inheritance calculations (Hanafi, Maliki, Shafi'i, Hanbali, Jafari)
- Heir selection and share distribution
- Professional PDF report generation
- Audit trail and calculation history
- Dark/light theme support
- Arabic and English language support
- Export to CSV and PDF
- Share calculation results

## Design Principles
- Islamic cultural sensitivity with green/blue/gold color palette
- Material Design 3 foundation
- Accessibility (WCAG AA compliant)
- Professional, non-AI-generated appearance
- Mobile-first responsive design

## Technical Stack
- React Native / Expo
- TypeScript
- Dexie (IndexedDB) for local storage
- i18next for internationalization
- Material Design 3 inspired components
