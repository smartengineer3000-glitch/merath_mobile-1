## 🎓 أمثلة سريعة

### مثال 1: حساب أساسي

**الورثة:** زوج + بنت  
**الحساب:** `1/2 + 1/2 = 1`  
**النتيجة:** توزيع متساو 50-50

```javascript
const engine = new InheritanceCalculationEngine(
  "shafii",
  { total: 100000, funeral: 0, debts: 0, will: 0 },
  { husband: 1, daughter: 1 },
);
const result = engine.calculate();
// husband: 50000, daughter: 50000
```

### مثال 2: حالة معقدة (عول)

**الورثة:** زوجة + بنتان + أب + أم  
**الحالة:** عول من 24 إلى 27  
**النتيجة:** توزيع متناسب

```javascript
const result = engine.calculate();
// يتم تطبيق معامل العول تلقائياً
// النتيجة: توزيع صحيح فقهياً
```

### مثال 3: اختبار

```javascript
const testSuite = new TestSuite();
await testSuite.runAllTests("shafii");

// النتائج: جميع الاختبارات تمر
```
