## 🔧 استخدام نظام الاختبارات

### 1. تشغيل الاختبارات البسيط:

```typescript
import { TestSuite } from "@/lib/inheritance";

const suite = new TestSuite(0.01); // 1% تسامح
const report = await suite.runAllTests();

console.log(report.summary);
// ✅ نجح: 11/11 اختبار (100%)
// ❌ فشل: 0
// ⏱️ الوقت الإجمالي: 45.23ms
```

### 2. تشغيل سريع:

```typescript
import { runTestSuite } from "@/lib/inheritance";

const report = await runTestSuite();
```

### 3. الوصول إلى النتائج:

```typescript
const results = report.results; // جميع نتائج الاختبارات
const failedTests = results.filter((r) => !r.passed);

for (const test of failedTests) {
  console.log(`❌ ${test.testId}: ${test.error}`);
}
```

### 4. حفظ التقرير:

```typescript
suite.saveReport(report, "test-report.json");
```
