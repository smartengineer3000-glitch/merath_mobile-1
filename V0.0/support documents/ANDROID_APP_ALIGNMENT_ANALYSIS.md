## المذاهب الأربعة والقواعس الفقهية

### 1️⃣ المذهب الشافعي (Shafi'i)

**الخصائص الرئيسية:**

| الحالة                | الحكم              | الملاحظات                  |
| --------------------- | ------------------ | -------------------------- |
| الجد مع الإخوة        | يحجبهم كلياً       | الجد لا يقاسم الإخوة       |
| الأم مع الأب والأبناء | السدس فقط          | لا تحصل على ثلث الباقي     |
| الرد على الزوج/الزوجة | لا يوجد            | الزوج/الزوجة لا يردّ عليهم |
| العمرية               | بتطبيق الحكم الأول | الأم = 1/3 الباقي          |

**قواعد الفروض:**

- الزوج: 1/2 أو 1/4
- الزوجة: 1/4 أو 1/8
- الأب: 1/6 أو عصبة
- الأم: 1/3 أو 1/6
- البنت: 1/2 أو 2/3
- الابن: عصبة
- الأخ الشقيق: عصبة

### 2️⃣ المذهب الحنفي (Hanafi)

**الخصائص الرئيسية:**

| الحالة                | الحكم      | الملاحظات               |
| --------------------- | ---------- | ----------------------- |
| الجد مع الإخوة        | يقاسمهم    | الجد يأخذ حصة أخ        |
| الأم مع الأب والأبناء | 1/3 الباقي | أكثر من الشافعي         |
| الرد على الزوج/الزوجة | يوجد رد    | ينطبق على الزوج والزوجة |
| الأب مع الأبناء       | عصبة       | إذا لم يأخذ الفرض       |

**قاعدة المقاسمة:**

- إذا كان الجد مع إخوة أشقاء، يختار الأفضل:
  - إما يأخذ الثلث (كجد)
  - أو يقاسمهم كأخ

### 3️⃣ المذهب المالكي (Maliki)

**الخصائص الرئيسية:**

| الحالة                | الحكم        | الملاحظات              |
| --------------------- | ------------ | ---------------------- |
| الجد مع الإخوة        | يقاسمهم      | المقاسمة الأساسية      |
| الأم مع الأب والأبناء | 1/6 فقط      | أقل من الشافعي والحنفي |
| غياب الورثة           | ميراث الدولة | نسبة من التركة         |
| الرد على الإخوة       | يوجد رد      | للإخوة والأخوات        |

### 4️⃣ المذهب الحنبلي (Hanbali)

**الخصائص الرئيسية:**

| الحالة                | الحكم              | الملاحظات          |
| --------------------- | ------------------ | ------------------ |

| الأم مع الأب والأبناء | 1/6 فقط            | حكم مالكي          |
| العمرية               | بتطبيق الحكم الأول | كالشافعي           |

### جدول 3: معادلات الحساب النهائي

```
التركة الصافية = إجمالي التركة - التجهيز - الديون

للورثة:
النصيب = (التركة الصافية × نسبة الفرض) / مجموع النسب

في حالة الوصية:
التركة المتاحة للورثة = (التركة الصافية × 2/3) - الوصايا

في حالة العول:
النسبة المعدلة = النسبة الأصلية × (المقام الأصلي / المقام الجديد)

في حالة الرد:
النسبة المعدلة = النسبة الأصلية × (التركة المتبقية / مجموع النسب)
```

## ملحق: مثال عملي كامل

### المثال: زوجة + أب + أم (العمرية الثانية)

#### البيانات المدخلة:

```javascript
{
  madhab: 'shafii',
  estate: {
    total: 120000,
    funeral: 0,
    debts: 0,
    will: 0
  },
  heirs: {
    wife: 1,
    father: 1,
    mother: 1
  }
}
```

#### الخطوات الحسابية:

**الخطوة 1: تطبيق الفروض**

- الزوجة: 1/4 (لأنه لا يوجد أبناء)
- الأب: 1/6 (فرض)
- الأم: 1/3 (نصيبها الأصلي)

**الخطوة 2: حساب المجموع**

- المجموع = 1/4 + 1/6 + 1/3
- = 3/12 + 2/12 + 4/12
- = 9/12 = 3/4

**الخطوة 3: تطبيق الرد (لأن المجموع < 1)**

- الباقي = 1 - 3/4 = 1/4

**الخطوة 4: توزيع الباقي على ذوي الفروض**

- الزوجة: 1/4 + (1/4 × 1/4 ÷ 3/4) = 25000 + 8333.33 = 33333.33
- الأب: 1/6 + (1/4 × 1/6 ÷ 3/4) = 20000 + 5555.56 = 25555.56
- الأم: 1/3 + (1/4 × 1/3 ÷ 3/4) = 40000 + 11111.11 = 51111.11

#### النتائج النهائية:

```
الزوجة:  33,333.33 (27.78%)
الأب:    25,555.56 (21.30%)
الأم:    51,111.11 (42.59%)
الخطأ التقريبي: 0%
```
