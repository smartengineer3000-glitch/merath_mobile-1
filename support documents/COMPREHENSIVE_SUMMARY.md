## 🎓 المذاهب الأربعة - الملخص

### 1. 🕌 المذهب الشافعي
**الخصائص:**
- الجد يحجب الإخوة (لا يقاسمهم)
- الأم = 1/6 مع الأب والأبناء
- لا رد على الزوج/الزوجة
- العمرية بتطبيق الحكم الأول

**الحالات المميزة:** 3

### 2. 📖 المذهب الحنفي
**الخصائص:**
- الجد يقاسم الإخوة
- الأم قد تأخذ أكثر
- رد على الزوج/الزوجة
- نسبة خاصة للمقاسمة

**الحالات المميزة:** 4

### 3. ⚖️ المذهب المالكي
**الخصائص:**
- الجد يقاسم الإخوة
- الأم = 1/6 دائماً مع الأب والأبناء
- ميراث الدولة ممكن
- رد على الورثة

**الحالات المميزة:** 3

### 4. 📜 المذهب الحنبلي
**الخصائص:**
- الجد يحجب الإخوة (كالشافعي)
- الأم = 1/6 مع الأب والأبناء
- رد محدود
- عمرية خاصة

**الحالات المميزة:** 3

## 🧮 أمثلة حسابية من HTML

### مثال 1: الحالة الأساسية
```
الورثة: زوج + بنت
المذهب: الشافعي
التركة: 120,000 ريال

الحساب:
- الزوج: 1/2 = 60,000 (50%)
- البنت: 1/2 = 60,000 (50%)

الحالات الخاصة: لا توجد
الثقة: 100%
```

### مثال 2: العمرية الثانية (المعقدة)
```
الورثة: زوجة + أب + أم
المذهب: الشافعي
التركة: 120,000 ريال

الخطوات:
1. الزوجة: 1/4 (لا يوجد أبناء)
2. الأب: 1/6 (فرض)
3. الأم: 1/3 (الأصلي) → 1/3 الباقي (مصحح)

المجموع: 1/4 + 1/6 + 1/3 = 3/4
الباقي: 1/4 → يُوزع على ذوي الفروض

النتيجة النهائية:
- الزوجة: ≈33,333 (27.78%)
- الأب: ≈25,556 (21.30%)
- الأم: ≈61,111 (50.92%)

الحالات الخاصة: رد
الثقة: 100%
```

### مثال 3: حالة العول
```
الورثة: زوجة + بنتان + أب + أم
المذهب: الشافعي
التركة: 120,000 ريال

الفروض الأصلية:
- الزوجة: 1/4
- البنتان: 2/3
- الأب: 1/6
- الأم: 1/6

المجموع: 1/4 + 2/3 + 1/6 + 1/6 = 27/24 (عول)

التطبيق:
- يتم تعديل كل نسبة بنسبة 24/27
- النتيجة النهائية موزعة بالتناسب

النتيجة: توزيع متناسب
الحالات الخاصة: عول من 24 إلى 27
الثقة: 100%
```
