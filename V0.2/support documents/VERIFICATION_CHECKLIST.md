## قائمة التحقق الوظيفية

### ✅ القسم الأول: محرك الحسابات (Calculation Engine)

#### A. الكسور والعمليات الحسابية

- [ ] **FractionClass.add()** - جمع كسرين
  - المتطلب: `1/4 + 1/6 = 5/12`
  - الاختبار: تجميع فروض متعددة
  - مستوى الأهمية: 🔴 حرجة

- [ ] **FractionClass.subtract()** - طرح كسرين
  - المتطلب: `1 - 1/4 = 3/4`
  - الاختبار: حساب الباقي
  - مستوى الأهمية: 🔴 حرجة

- [ ] **FractionClass.multiply()** - ضرب الكسر برقم
  - المتطلب: `1/6 × 2 = 1/3`
  - الاختبار: توزيع على عدة ورثة
  - مستوى الأهمية: 🔴 حرجة

- [ ] **FractionClass.divide()** - قسمة الكسر على رقم
  - المتطلب: `1/2 ÷ 2 = 1/4`
  - الاختبار: توزيع متساو
  - مستوى الأهمية: 🔴 حرجة

- [ ] **FractionClass.toDecimal()** - تحويل إلى عدد عشري
  - المتطلب: دقة ≥ 10 منازل عشرية
  - الاختبار: حساب المبالغ النهائية
  - مستوى الأهمية: 🔴 حرجة

- [ ] **FractionClass.simplify()** - تبسيط الكسر
  - المتطلب: `2/4 → 1/2`
  - الاختبار: تطبيق تلقائي بعد كل عملية
  - مستوى الأهمية: 🟡 هامة

#### B. الحجب الفقهي

- [ ] **Hijab: Son blocks siblings**
  - المتطلب: الابن يحجب الإخوة والأخوات
  - الاختبار: حالة (ابن + إخوة أشقاء)
  - النتيجة المتوقعة: الإخوة = 0
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Hijab: Father blocks grandfather**
  - المتطلب: الأب يحجب الجد
  - الاختبار: حالة (أب + جد + ابن)
  - النتيجة المتوقعة: الجد = 0
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Hijab: Mother blocks grandmother**
  - المتطلب: الأم تحجب الجدة
  - الاختبار: حالة (أم + جدة + ابن)
  - النتيجة المتوقعة: الجدة = 0
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Partial Hijab: Father with descendants**
  - المتطلب: الأب يأخذ 1/6 فقط بدلاً من التعصيب
  - الاختبار: حالة (أب + ابن)
  - النتيجة المتوقعة: الأب = 1/6
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Binna Hijab: Daughter becomes co-heir**
  - المتطلب: البنت الوحيدة = 1/2، مع أخت = 2/3
  - الاختبار: إضافة ثانية يغير النسبة
  - النتيجة المتوقعة: تغيير الفرض
  - مستوى الأهمية: 🔴 حرجة

#### C. الفروض الشرعية

- [ ] **Provision: Husband (no children)**
  - المتطلب: الزوج = 1/2
  - الاختبار: (زوج + بنت)
  - النتيجة المتوقعة: زوج = 1/2، بنت = 1/2
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Husband (with children)**
  - المتطلب: الزوج = 1/4
  - الاختبار: (زوج + ابن)
  - النتيجة المتوقعة: زوج = 1/4، ابن = 3/4
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Wife (no children)**
  - المتطلب: الزوجة = 1/4
  - الاختبار: (زوجة + ابن)
  - النتيجة المتوقعة: زوجة = 1/4، ابن = 3/4
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Wife (with children)**
  - المتطلب: الزوجة = 1/8
  - الاختبار: (زوجة + ابن)
  - النتيجة المتوقعة: زوجة = 1/8، ابن = 7/8
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Father with descendants**
  - المتطلب: الأب = 1/6 (فرض) + الباقي (تعصيب)
  - الاختبار: (أب + بنت)
  - النتيجة المتوقعة: أب = 1/3، بنت = 2/3 (بعد الفرض)
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Mother (basic)**
  - المتطلب: الأم = 1/3 (بدون أبناء)
  - الاختبار: (أم وحدها)
  - النتيجة المتوقعة: أم = 1
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Provision: Mother with Umariyyah**
  - المتطلب: الأم = 1/3 الباقي (بعد فروض الزوج والأب)
  - الاختبار: (زوج + أب + أم)
  - النتيجة المتوقعة: حساب خاص
  - مستوى الأهمية: 🔴 حرجة

#### D. الحالات الخاصة

- [ ] **Awl (النعول)** - جمع الفروض > التركة
  - المتطلب: تطبيق النسبة المعدلة
  - الاختبار: (زوجة + بنتان + أب + أم)
  - النتيجة المتوقعة: توزيع متناسب
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Radd (الرد)** - جمع الفروض < التركة
  - المتطلب: توزيع الباقي على ذوي الفروض
  - الاختبار: (أم + بنت)
  - النتيجة المتوقعة: توزيع الباقي بنسبة 1/6 : 1/2
  - مستوى الأهمية: 🔴 حرجة

- [ ] **Umariyyah (العمرية)**
  - المتطلب: معالجة خاصة للأم مع الأب والزوج/الزوجة
  - الاختبار: (زوجة + أب + أم)
  - النتيجة المتوقعة: نتيجة محددة حسب المذهب
  - مستوى الأهمية: 🔴 حرجة

### ✅ القسم الثاني: المذاهب الأربعة

#### Shafi'i School (الشافعي)

- [ ] **Grandfather blocks siblings**
  - المتطلب: الجد يحجب الإخوة كلياً
  - الاختبار: (جد + إخوة أشقاء)
  - النتيجة المتوقعة: جد = كل شيء، إخوة = 0
  - المقارنة مع غيره: فريد الشافعي

- [ ] **Mother with father and descendants**
  - المتطلب: الأم = 1/6 فقط
  - الاختبار: (أب + أم + ابن)
  - النتيجة المتوقعة: نسبة محددة
  - المقارنة مع غيره: تختلف عن الحنفي

- [ ] **No Radd on spouse**
  - المتطلب: الزوج/الزوجة لا يستفيدان من الرد
  - الاختبار: (زوج وحده) أو (زوجة وحدها)
  - النتيجة المتوقعة: كل شيء للزوج/الزوجة
  - المقارنة مع غيره: يختلف عن الحنفي

#### Hanafi School (الحنفي)

- [ ] **Grandfather shares with siblings**
  - المتطلب: الجد يقاسم الإخوة
  - الاختبار: (جد + إخوة أشقاء)
  - النتيجة المتوقعة: جد = 1/x، إخوة = (x-1)/x
  - المقارنة مع غيره: يختلف عن الشافعي

- [ ] **Mother gets more**
  - المتطلب: الأم قد تأخذ أكثر من الشافعي
  - الاختبار: (أب + أم)
  - النتيجة المتوقعة: نسبة مختلفة
  - المقارنة مع غيره: أكثر سخاء للأم

- [ ] **Radd on spouse**
  - المتطلب: الزوج/الزوجة يستفيدان من الرد
  - الاختبار: (زوج وحده)
  - النتيجة المتوقعة: زوج = 1 (مع الرد)
  - المقارنة مع غيره: فريد الحنفي

#### Maliki School (المالكي)

- [ ] **Grandfather shares with siblings**
  - المتطلب: الجد يقاسم الإخوة (كالحنفي)
  - الاختبار: (جد + إخوة أشقاء)
  - النتيجة المتوقعة: توزيع متساو
  - المقارنة مع غيره: مشابه للحنفي

- [ ] **Mother gets least**
  - المتطلب: الأم تأخذ الأقل بين المذاهب
  - الاختبار: (أب + أم)
  - النتيجة المتوقعة: الأم = 1/6
  - المقارنة مع غيره: أقل سخاء للأم

#### Hanbali School (الحنبلي)

- [ ] **Grandfather blocks siblings**
  - المتطلب: الجد يحجب (كالشافعي)
  - الاختبار: (جد + إخوة أشقاء)
  - النتيجة المتوقعة: جد = كل شيء
  - المقارنة مع غيره: مشابه للشافعي

- [ ] **Mother treatment**
  - المتطلب: تطبيق خاص للأم
  - الاختبار: حالات متعددة
  - النتيجة المتوقعة: نسب محددة
  - المقارنة مع غيره: متوازن
